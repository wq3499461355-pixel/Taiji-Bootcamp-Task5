### 1. 空间引力波探测的数据长什么样？
- 空间探测器（比如太极、LISA）由三颗卫星组成，它们用激光互相照射，测量彼此之间的距离变化。
- 引力波经过时，会拉伸或压缩空间，激光测距就会产生微小的波动。
- 这些波动记录下来就是**时间序列数据**——就是一串随时间变化的数字。数据里混杂着引力波信号、仪器的噪音、以及很多其他天体产生的干扰（比如银河系里无数的双白矮星）。

### 2. 为什么要做傅里叶变换？
- 原始数据是随时间变化的（时域），但引力波信号的特征在频率上更明显（比如某个频率范围内信号更强）。
- 傅里叶变换就像一台“频率分解机”，把时间序列拆解成不同频率的成分。这样我们可以更容易地把信号从噪音里分出来。
- 就好比一首歌里有人声、鼓点、吉他，傅里叶变换能帮你分别看到每种声音的频率分布。

### 3. 什么是大质量双黑洞并合信号？
- 两个质量很大的黑洞（比如太阳质量的几万倍甚至几百万倍）互相绕转、最后撞在一起，会产生引力波。
- 这种并合发出的引力波频率很低，正好是空间探测器（如太极）的“听觉范围”。地面探测器（LIGO）听不到这么低频的信号。
- 这个信号的样子取决于黑洞的质量、自旋、轨道倾角、距离等等。我们用数学模型（波形模板）来描述它。

### 4. 贝叶斯统计推断是干嘛的？
- 简单说：**用观测数据，反过来猜源头参数**。
- 我们手里有一串数据（包含信号+噪音），想知道黑洞的质量到底是多少。贝叶斯推断会给出一个**概率分布**，而不是一个单一数字。比如：“质量有90%的概率在 30万到 35万太阳质量之间”。
- 这种推断方法能处理不确定性，还能看出哪些参数之间互相牵连（比如距离和倾角往往分不清）。
- 这个任务里用的**嵌套采样**就是一种贝叶斯推断算法，它像爬楼梯一样，从大概率区域慢慢探索到小概率区域，最后画出一张**corner图**（就是那种很多小图拼成的大图），直观展示每个参数的可能范围和参数之间的关联。
